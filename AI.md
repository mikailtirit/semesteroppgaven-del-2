# AI.md

## Use of Artificial Intelligence in This Project

This document explains how Artificial Intelligence (AI) was used during the development of **ChickFiley Movies**.

The purpose of this file is to be transparent and honest about the development process, while also explaining what I learned throughout the project.

---

# AI Tools Used

Several AI tools were used during different parts of the project:


- **ChatGPT** – coding help, debugging, structure, explanations, feature building, UI/UX improvements  
- **Claude** – coding support, writing assistance, idea generation, alternative solutions  
- **Gemini** – research help, brainstorming, technical explanations, feature suggestions  
- **Grok** – image creation and visual asset generation

Each tool was used as support depending on the task.

---

# Overall Use of AI

A large portion of the code in this project was created with the help of AI assistance.

AI was used frequently during development to generate code examples, suggest solutions, improve structure, solve bugs, and help build new features faster.

This includes support with:

- Flask routes
- Authentication logic
- MySQL queries
- API integration
- HTML templates
- CSS styling
- Debugging
- UI / UX improvements
- Code cleanup and refactoring

---

# Why I Used AI

I used AI as a tool to help me learn, improve workflow, and understand concepts that were new or difficult.

Instead of getting stuck for long periods of time, AI helped explain problems and provide examples that I could test, study, and build on.

It also helped me create a more advanced final project than I would likely have achieved alone.

---

# What I Did Myself

Even though AI helped generate much of the code, I was still fully involved in the development process and understand how the project works overall.

I personally:

- Decided what type of website to build
- Planned the features and user experience
- Chose the design direction and layout
- Tested the application continuously
- Implemented and connected different parts together
- Adjusted and customized generated code when needed
- Fixed bugs and errors during development
- Structured the project files and folders
- Improved the design through my own ideas
- Made final decisions on all features and changes

Most importantly, I understand the overall logic of the application, how the backend works, how the database is connected, how routes function, and how the frontend is rendered.

I did not simply copy code blindly — I used the project as a learning process and made sure I understood what was being built.

---

# What I Learned

Using AI during this project helped me learn a lot about real development.

I improved my understanding of:

- Python and Flask
- SQL databases
- User authentication systems
- Sessions and security
- API requests and JSON data
- HTML / CSS structure
- UI / UX design principles
- Debugging techniques
- How full-stack applications are built

Most importantly, I learned how to read, understand, modify, and apply code instead of only writing it manually.

---

# Reflection

AI was an important part of this project and contributed significantly to the coding process.

At the same time, I used the project as a learning experience by studying the generated code, asking questions, testing solutions, and improving the final result through my own decisions.

This showed me that AI can be a powerful tool when used responsibly as support during development.

---

# Final Statement

Much of the code in this project was created with AI assistance, but the direction of the project, implementation process, feature decisions, testing, and learning process were all actively handled by me.

I understand the codebase overall and how the different systems work together.

AI was used as a development tool — not as a replacement for learning.
